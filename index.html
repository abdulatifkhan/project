Bu Git va GitHub bilan ishlash uchun birinchi loyiha

  1. git init - loyihamizda git texnologiyasini ishlatib yuborish;
  2. git status - loyihamizni holatini ko'rib olish uchun;
  3. git add . - hamasini bittada joylash uchun tayyor qilib qo'yish;
  4. git commit -m "Qilgan ishimizni izohi, ingliz tilida, Present Simple zamonida";
  5. git push - loyihamiz qismini GitHub profilimizda ochilgan repo'ga jo'natish.
